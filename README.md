# Landio – Responsive Landing Page

Landio is a modern and responsive landing page template built using HTML and CSS.  
The project focuses on clean UI design, responsive layouts, and well-structured front-end code.

## 🚀 Features

- Fully responsive design
- Modern and clean UI
- Built with CSS Grid and Flexbox
- Smooth animations using AOS
- Organized and maintainable code structure
- Cross-browser compatible

## 🛠️ Technologies Used

- HTML5
- CSS3
- AOS (Animate On Scroll)
- Font Awesome
- Bootstrap Icons

## 📱 Responsive Design

The layout is fully responsive and works on:

- Desktop
- Tablet
- Mobile devices
